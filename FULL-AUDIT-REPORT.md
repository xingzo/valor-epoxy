### Top 5 Critical Issues

1. **Terrible URL slugs** — Pages use `/blank-1`, `/blank-2`, `/blank-3`, `/blank-4` instead of `/about`, `/quote`, `/services`, `/contact`. Portfolio projects use `/project-title-1` through `/project-title-6`. These generic URLs destroy keyword signals and confuse both users and search engines.
2. **No H1 tag on homepage** — The most important page on the site has no H1 heading element, which is a primary ranking signal.
3. **No blog or content strategy** — Zero blog posts, no educational content, no FAQ page. For a local service business, content is essential for ranking on long-tail keywords.
4. **Minimal structured data** — Only basic LocalBusiness and WebSite schema on the homepage. No Service, FAQ, BreadcrumbList, or Review schema anywhere.
5. **Thin content on key pages** — Services page uses generic placeholder text. Portfolio page and contact page have almost no indexable content.

### Top 5 Quick Wins

1. **Rename page URLs** from `/blank-1` etc. to descriptive slugs (`/about`, `/services`, `/contact`, `/quote`)
2. **Add an H1 tag** to the homepage (e.g., "Dallas-Fort Worth Epoxy Flooring & Concrete Coatings")
3. **Rename portfolio project URLs** from `project-title-X` to descriptive names matching each project
4. **Expand LocalBusiness schema** with areaServed, priceRange, openingHours, description, and geo coordinates
5. **Add schema markup** to inner pages (Service schema on services page, BreadcrumbList on all pages)

### Wix Platform Limitations

Since the site runs on Wix, many performance optimizations are outside the owner's control:
- Cannot minify or defer custom scripts
- Cannot implement custom caching strategies
- Cannot use a custom CDN
- HTML bloat from Wix's rendering engine is unavoidable
- Core Web Vitals optimization is limited to Wix's built-in capabilities

### Estimated Performance Impact

- **LCP (Largest Contentful Paint):** Likely poor due to heavy HTML and hero image
- **INP (Interaction to Next Paint):** Likely moderate — JavaScript heavy
- **CLS (Cumulative Layout Shift):** Likely acceptable — Wix handles layout reservations

> Note: Actual Core Web Vitals were not measured with Lighthouse in this audit. The above are estimates based on page analysis. Recommend running PageSpeed Insights for precise measurements.
